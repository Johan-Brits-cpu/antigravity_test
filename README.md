# Pomodoro Timer

A modern, aesthetic Pomodoro timer web application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Visual Timer**: A beautiful SVG-based progress ring that allows you to visualize the remaining time.
- **Customizable Duration**: Set your own focus time in minutes.
- **Sound Notifications**: Plays a gentle alarm sound when the timer completes.
- **Dark Mode Design**: sleek, dark-themed interface for reduced eye strain.
- **Responsive Controls**: Easy-to-use Start, Pause, and Reset buttons.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox, and transitions for animations.
- **JavaScript (ES6+)**: `AudioContext` for sound, class-based state management.

## How to Run

1.  Clone this repository or download the files.
2.  Open `index.html` in your web browser.
3.  Set your desired time (default is 25 minutes) and click the "Play" button to focus!
